SCA Academia is a complete ERP platform built for coaching academies. It brings together student enrollment, fee collection with automatic 70/30 revenue splitting, barcode-based gatekeeper entry, teacher payroll, and a self-service student portal — all in one cohesive system.

## Get started in four steps

<Steps>
  <Step title="Install dependencies">
    Clone the repository and install both backend and frontend packages.
  </Step>
  <Step title="Configure environment">
    Set your MongoDB URI and JWT secret in `backend/.env`.
  </Step>
  <Step title="Seed the database">
    Run `npm run seed` to create the initial owner account and sample data.
  </Step>
  <Step title="Log in">
    Open the frontend, sign in as the OWNER, and explore your academy dashboard.
  </Step>
</Steps>

<Note>
  The full setup walkthrough — including environment variables and first login — is in the [Quick start](/quickstart) guide.
</Note>
