SCA Academia is an end-to-end ERP built specifically for coaching academies. It replaces paper registers and spreadsheets with a unified system that tracks every student, rupee, and staff member from a single dashboard. The platform is used daily by owners, financial partners, and front-desk staff across all operational areas of the academy.

<Note>
  SCA Academia uses a three-tier role model: **OWNER** has full system access including expense distribution and staff management; **PARTNER** sees only their own revenue stream and can perform daily closings; **STAFF/OPERATOR** has operational access only with no financial visibility. All roles are enforced at the API level via JWT middleware.
</Note>

## Tech Stack

The system is built on the **MERN stack** with a clear separation between backend services and the frontend application.

**Backend**
- **Node.js + Express 5** — HTTP server and REST API
- **MongoDB + Mongoose 9** — Document database and ODM
- **jsonwebtoken + bcryptjs** — Authentication and password hashing
- **cookie-parser** — HTTP-only cookie handling
- **multer** — File and image uploads (up to 50 MB)

**Frontend**
- **React 18 + Vite** — Component framework and build tool
- **TypeScript** — Type-safe component authoring
- **Tailwind CSS + shadcn/ui (Radix UI)** — Design system and accessible primitives
- **TanStack Query (React Query v5)** — Server-state caching and synchronization
- **React Router v6** — Client-side routing
- **React Hook Form + Zod** — Form handling and validation
- **Recharts** — Dashboard charts and financial graphs
- **@react-pdf/renderer** — In-browser PDF generation for receipts and payslips
